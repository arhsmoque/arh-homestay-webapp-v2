# Responsive Layout Rules

## Mobile

- Hero image first, then headline and CTA.
- Proof strip as 2-column compact grid.
- Occasion, package, room, event cards stacked single column.
- Gallery as large story cards, not tiny thumbnails.
- Sticky bottom CTA: WhatsApp / Map / Check Date.
- No hover-only content.
- No tiny icon-only controls.
- No dense pricing tables.

## Tablet

- Occasion cards in 2-column grid.
- Room cards in 2-column grid.
- Package cards in 2-column or horizontal scroll.
- Layout preview may remain stacked.

## Desktop

- Hero split layout: content left, media right.
- Proof strip single row.
- Occasion cards 4-card grid.
- Pool feature large image with facts panel.
- Rooms: sleep summary + room grid.
- Layout preview: map left, explanation right.
- Packages: 3-card comparison row.
- FAQ: accordion or two-column preview.

## Rhythm

```text
Big emotional section
→ compact fact section
→ choice cards
→ proof media
→ detail cards
→ CTA
```
