# Guest Journeys

## Journey 1 — Google Search to WhatsApp

```text
Search:
"homestay private pool near [area]"

Landing:
Home or Private Pool page

Need:
Pool proof, capacity, location, availability path

Conversion:
Sticky WhatsApp CTA with structured message
```

## Journey 2 — TikTok to Property Proof

```text
TikTok video:
Pool/night/rain/family-day clip

Landing:
Private Pool, Events, Layout, or Location page

Need:
Fast confirmation that the place is real, suitable, and bookable

Conversion:
Tap WhatsApp / save / share / check available dates
```

## Journey 3 — Event Organizer to Layout Proof

```text
Search/social:
"private event space for family day"
"homestay for corporate retreat"
"small event space with pool"

Landing:
Events page

Need:
Layout, pax, parking, rain plan, package suitability

Conversion:
Event-specific WhatsApp inquiry
```

## Journey 4 — Outstation / International Guest to Trust Proof

```text
Search:
"family villa near [landmark]"
"private pool homestay Malaysia"

Landing:
Location + Stay page

Need:
Map, distance, facilities, check-in, rules, clear English

Conversion:
Inquiry with date, pax, purpose, and location questions
```
