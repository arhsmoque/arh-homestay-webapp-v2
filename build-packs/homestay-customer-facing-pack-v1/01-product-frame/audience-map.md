# Audience Map

## Local Family Staycation

```yaml
primary_question: "Can my family enjoy this place comfortably?"
sources:
  - Google Search
  - Google Maps
  - Facebook
  - TikTok
  - WhatsApp share
decision_signals:
  - private pool
  - bedrooms and sleeping arrangement
  - kitchen
  - BBQ
  - parking
  - nearby shops and food
  - child-friendly practical signals
anxieties:
  - "Will it look like the photos?"
  - "Is the pool really private?"
  - "Can everyone sleep comfortably?"
  - "Is parking easy?"
```

## Extended Family / Family Day

```yaml
primary_question: "Can this place handle our group without feeling cramped?"
decision_signals:
  - event pax
  - seating / eating zones
  - pool visibility
  - gazebo / pergola
  - rain plan
  - parking count
  - rules on noise and time
anxieties:
  - "Where will people gather?"
  - "Where will cars go?"
  - "What if it rains?"
  - "Can kids and elders both be comfortable?"
```

## Small Corporate / SME Retreat

```yaml
primary_question: "Is this professional enough for a relaxed private event?"
decision_signals:
  - private compound
  - common area
  - seating setup
  - lunch / BBQ support
  - parking
  - quiet environment
  - optional setup
anxieties:
  - "Is it too chalet-like?"
  - "Will it feel premium enough?"
  - "Can we run discussion and bonding in one place?"
```

## Ceremony and Private Event Users

```yaml
primary_question: "Can this private place host a controlled family event?"
decision_signals:
  - photo spots
  - covered areas
  - event flow
  - parking
  - family privacy
  - food setup
  - toilet access
anxieties:
  - "Will the layout support guests?"
  - "Can we decorate?"
  - "Where do people sit?"
  - "Can older family members move around comfortably?"
```

## International / Outstation Guests

```yaml
primary_question: "Can I trust this place and understand the logistics?"
decision_signals:
  - exact location
  - landmarks
  - airport / town / hospital distance
  - check-in process
  - house rules
  - Wi-Fi
  - owner contact
anxieties:
  - "Is this legitimate?"
  - "How do I reach the place?"
  - "Will communication be easy?"
  - "Is the listing information complete?"
```
