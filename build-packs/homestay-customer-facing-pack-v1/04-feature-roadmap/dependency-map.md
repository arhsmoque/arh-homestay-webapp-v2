# Feature Dependency Map

```text
Property facts
  → proof cards
  → structured pages
  → WhatsApp payload
  → SEO metadata
  → future AI assistant

Media registry
  → grouped gallery
  → hero media
  → social previews
  → TikTok landing sections
  → future media admin

Package model
  → package cards
  → inquiry type
  → future pricing
  → future booking/payment

Availability model
  → check date CTA
  → owner blocking
  → public calendar
  → payment/deposit flow

Rules / FAQ model
  → FAQ page
  → booking confidence
  → legal display
  → future AI assistant

Analytics events
  → WhatsApp clicks
  → map clicks
  → page performance
  → campaign tracking
  → future owner dashboard
```

Rule:

```text
Do not build advanced features before the data model exists.
```
