# Google Maps Strategy

## Role

Google Maps is a trust and arrival checkpoint.

## Website Components

- LocationProofCard
- NearbyLandmarkCard
- RouteProofCard
- GoogleMapsCta
- MapCtaCard

## Required Public Content

- Google Maps link
- Address or area-level address according to owner preference
- Coordinates if used for structured data
- Road name / landmark
- Gate or entrance photo
- Parking proof
- Nearby places
- Route video if available

## CTA Labels

- Open Google Maps
- Copy address
- Send location by WhatsApp
- View nearby places

## Location Page Structure

```text
/location
  Hero: Easy to reach near [Landmark]
  Map embed or map card
  Open Google Maps CTA
  Nearby cards
  Arrival instructions
  Gate / parking photo
  Route video
  FAQ: parking, road access, landmark, check-in
```
