# SEO Checklist

## Required

- Unique title per page.
- Unique meta description per page.
- Canonical URLs.
- Sitemap.
- Robots.txt.
- Open Graph image.
- Social preview title and description.
- Image alt text.
- Clean route slugs.
- Page-specific search intent.

## Structured Data Scaffold

- Property name.
- Description.
- Images.
- Address or location.
- Coordinates.
- Occupancy.
- Rooms / beds / bathrooms.
- Amenities.
- Check-in / check-out if available.
- Review/rating fields only if valid and permitted.
