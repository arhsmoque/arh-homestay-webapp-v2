# Performance Checklist

## Core Rules

- Optimize hero image.
- Reserve image and video dimensions.
- Lazy-load below-fold gallery.
- Avoid heavy autoplay.
- Hydrate only interactive islands.
- Defer analytics where possible.
- Compress CSS/JS.
- Avoid unnecessary third-party embeds.
- Use thumbnails for videos.
- Keep first screen lightweight.

## Public MVP Rejects

- Full social feed embeds above fold.
- Multiple autoplay videos.
- Heavy public SPA shell for mostly content pages.
- Uncompressed large photos.
- Layout shift from missing image dimensions.
