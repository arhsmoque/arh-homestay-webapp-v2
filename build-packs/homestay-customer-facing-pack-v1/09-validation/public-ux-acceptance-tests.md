# Public UX Acceptance Tests

## 3-Second Test

A first-time visitor can answer:

- What is this?
- Where is it?
- What is the main hook?
- What should I tap next?

## 30-Second Test

A serious visitor can answer:

- How many pax can sleep?
- Is the pool private?
- How many cars can park?
- Is it suitable for family day / event?
- Where is it?
- How do I ask for availability?

## Event-User Test

An event visitor can answer:

- What event types are suitable?
- What is the event pax range?
- Where do people gather?
- Where do cars park?
- What happens if it rains?
- What rules apply?

## Trust Test

Every major claim has proof through:

- photo
- map
- layout
- review
- FAQ
- rule
- hard fact

## No-Bloat Test

The MVP does not require:

- login
- payment
- full calendar
- AI chatbot
- 3D walkthrough

to send a useful inquiry.
