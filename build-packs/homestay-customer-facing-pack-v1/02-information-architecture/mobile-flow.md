# Mobile Flow

Mobile is the primary customer surface.

## Sticky Actions

```text
[WhatsApp] [Map] [Check Date]
```

## Scroll Priority

1. Hero and CTA
2. Proof strip
3. Best-for cards
4. Pool
5. Pax / rooms
6. Layout
7. Events
8. Location
9. Packages
10. FAQ / rules
11. Final inquiry

## Mobile Rules

- Use large tap targets.
- Avoid hover-only interactions.
- Keep proof chips short.
- Use single-column cards for occasion and packages.
- Lazy-load gallery images below fold.
- Keep WhatsApp CTA always reachable but not blocking content.
- Avoid autoplay video-heavy first screen.
