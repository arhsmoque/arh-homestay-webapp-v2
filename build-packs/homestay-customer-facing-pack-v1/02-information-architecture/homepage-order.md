# Homepage Order

The homepage should follow the decision sequence:

```text
Desire
→ Fit
→ Proof
→ Logistics
→ Occasion
→ Trust
→ Action
```

## Section Order

1. Hero Decision Card
2. Fast Proof Strip
3. Best-For Occasion Cards
4. Signature Private Pool Section
5. Sleep / Stay Fit Section
6. Compound Layout Preview
7. Event Suitability Preview
8. Gallery Story Preview
9. Location / Nearby Preview
10. Package Summary Cards
11. Review / Social Proof
12. FAQ / Rules Preview
13. Final WhatsApp Inquiry CTA

## Above the Fold

```text
[Hero image or short muted loop]
Private Pool Homestay near [Location]
For family stays, BBQ, family day, small events, and relaxed private gatherings.

[Private Pool] [Sleeps X] [Parking X Cars] [Near Landmark]

[Check Availability] [View Layout]
```

Do not waste first-screen space on generic welcome copy, decorative slogans, logo animation, or a dense navigation menu.
