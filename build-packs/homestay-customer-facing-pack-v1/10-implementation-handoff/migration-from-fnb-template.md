# Migration from F&B Template

## Brownfield Posture

The current F&B-derived site is a useful prototype, not a final metaphor.

```text
Salvage primitives.
Rewrite meaning.
```

## Salvage

- Cloudflare deployment workflow.
- Responsive primitives.
- Tailwind/shadcn components.
- WhatsApp CTA behavior.
- Useful media and visual assets.
- Any admin/content ideas for the operator pack.

## Rewrite

- Public page map.
- Homepage order.
- Card taxonomy.
- Booking purpose model.
- Package model.
- Property facts schema.
- Gallery grouping.
- Copy registers.
- Search/social metadata.

## Retire

- Menu/order mental model.
- Food item card hierarchy.
- Product variant assumptions.
- Food-commerce density.
- Cafe social tone.
- Cart metaphor unless carefully repurposed.

## Validation

The migrated site should pass:

- 3-second test.
- 30-second fit test.
- event-user test.
- WhatsApp inquiry test.
- mobile performance test.
- SEO/share preview test.
