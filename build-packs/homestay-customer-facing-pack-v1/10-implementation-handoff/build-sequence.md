# Build Sequence

## Phase 1 — Foundation

1. Create Astro project.
2. Add Tailwind/design tokens.
3. Create page routes.
4. Create content files.
5. Add metadata generator.
6. Add media registry.

## Phase 2 — Public Decision Spine

7. Build homepage.
8. Build Stay, Pool, Events, Layout, Location pages.
9. Build Gallery and Packages.
10. Build FAQ and Book pages.

## Phase 3 — Conversion

11. Build BookingModeSelector.
12. Build WhatsAppPayloadBuilder.
13. Add sticky mobile CTA.
14. Add package/event-specific WhatsApp messages.
15. Add fallback copy message.

## Phase 4 — Discovery

16. Add sitemap.
17. Add Open Graph images.
18. Add Google Maps CTA.
19. Add structured-data scaffold.
20. Add analytics event registry.

## Phase 5 — Quality

21. Optimize images.
22. Validate mobile layout.
23. Run accessibility checks.
24. Run performance checks.
25. Validate SEO and social preview.
26. Test WhatsApp payloads.
