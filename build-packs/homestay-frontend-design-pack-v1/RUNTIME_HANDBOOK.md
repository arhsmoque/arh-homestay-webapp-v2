# Runtime Handbook

Use this when debugging, reviewing, or modifying the frontend design under pressure.

## One-sentence doctrine

```text
Warm private retreat, proven by real media, guided by calm clarity.
```

## Do this first

```text
Before changing style, identify the role:
desire, fit, proof, logistics, occasion, trust, or action.
```

## Fast decision rules

```yaml
fast_rules:
  color:
    - pool_teal_is_identity
    - deep_pool_is_action
    - clay_is_event
    - olive_is_layout_nature
    - dusk_is_evening
    - warm_white_is_canvas

  typography:
    - Fraunces_for_emotion
    - Manrope_for_clarity
    - critical_facts_never_micro_text

  media:
    - real_photo_first
    - no_invented_property_features
    - gallery_grouped_by_purpose
    - video_poster_first

  motion:
    - use_motion_for_presence_not_showoff
    - reduced_motion_must_work
    - no_scroll_hijack

  components:
    - every_card_has_decision_role
    - clickable_cards_lift
    - nonclickable_cards_stay_calm

  booking:
    - WhatsApp_path_clear
    - fallback_copy_available
    - booking_form_focus_safe
```

## Safe edits

```yaml
safe_edits:
  - change copy
  - replace owner-approved photos
  - reorder featured gallery group
  - update package text
  - update FAQ/rules
  - choose hero media from approved registry
```

## Developer edits only

```yaml
developer_edits_only:
  - raw color values
  - font scale
  - motion profile
  - component variants
  - booking form required fields
  - accessibility behavior
  - responsive breakpoints
  - media loading strategy
```

## Red flags

```yaml
red_flags:
  - random hex values inside components
  - autoplay video grid
  - black-gold generic luxury
  - no focus state
  - hover-only detail
  - ungrouped gallery
  - tiny mobile tap targets
  - hero carousel before proof
  - owner editing raw design tokens directly
```

## Emergency simplification

When unsure, simplify to:

```text
real hero image
clear headline
proof chips
primary WhatsApp CTA
grouped gallery
layout proof
FAQ/rules
fast mobile performance
```
