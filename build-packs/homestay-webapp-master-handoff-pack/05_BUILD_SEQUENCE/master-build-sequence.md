# Master Build Sequence

## Phase 1 — Public foundation

```text
1. Create public Astro site.
2. Add Tailwind/design tokens.
3. Create public page routes.
4. Create property facts/content files.
5. Add media registry.
6. Build homepage decision spine.
7. Build Stay, Pool, Events, Layout, Location pages.
8. Build Packages, FAQ, and Book pages.
9. Add WhatsApp payload builder.
10. Add metadata, sitemap, Open Graph, and analytics events.
```

## Phase 2 — Operator control foundation

```text
11. Define roles and permissions.
12. Build audit event schema.
13. Implement mutation risk classes.
14. Implement dangerous action policy.
15. Build owner/developer route shell.
```

## Phase 3 — Booking engine

```text
16. Implement inquiry schema.
17. Implement booking schema.
18. Implement booking state machine.
19. Implement calendar block model.
20. Implement conflict checks.
21. Implement payment/deposit record model.
```

## Phase 4 — Owner dashboard

```text
22. Build OwnerTodayPage.
23. Build NeedsAttentionCard.
24. Build BookingPipelinePage.
25. Build BookingDetailPage.
26. Build BookingCalendarPage.
```

## Phase 5 — Commercial engine

```text
27. Build package model.
28. Build rate table.
29. Build quote builder.
30. Build public price preview.
31. Build owner override and rate versioning.
32. Build commercial audit trail.
```

## Phase 6 — Content/media/admin

```text
33. Build PropertyContentEditor.
34. Build MediaManager.
35. Build public preview before publish.
36. Build media validation and permission status.
```

## Phase 7 — Legal/privacy/rules

```text
37. Build RulesPolicyOverview.
38. Build policy editor.
39. Build policy versioning.
40. Build privacy notice model.
41. Link policy versions to bookings.
42. Build data retention/export warnings.
```

## Phase 8 — Analytics/observability/recovery

```text
43. Build owner analytics.
44. Build analytics event registry.
45. Build data health checks.
46. Build developer observability.
47. Build audit trail.
48. Build recovery controls.
49. Build handover export.
```
