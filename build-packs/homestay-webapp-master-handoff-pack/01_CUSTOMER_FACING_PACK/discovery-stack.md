# Discovery Stack

```text
Google finds.

Maps verifies.
Website proves.

Reviews compound.
Analytics learns.
```

## Channel Duties

```yaml
google_search:
  job: capture high-intent search
  site_response:
    - intent pages
    - page titles
    - meta descriptions
    - structured property facts
    - sitemap

google_maps:
  job: prove location and legitimacy
  site_response:
    - map CTA
    - nearby landmark cards
    - route proof
    - gate/parking photos

tiktok:
  job: create desire
  site_response:
    - pool landing page
    - event landing page
    - route landing page
    - 9:16 video slots

instagram_facebook:
  job: support sharing and social proof
  site_response:
    - Open Graph images
    - shareable pages
    - curated social proof

whatsapp:
  job: convert inquiry
  site_response:
    - sticky CTA
    - structured payload
    - purpose-specific messages

reviews:
  job: compound trust
  site_response:
    - review cards
    - Google review link
    - guest-type review grouping
```
