# Owner Control Panel Map

```text
Owner Control Panel
│
├─ Today
│  ├─ Needs Attention
│  ├─ New Inquiries
│  ├─ Pending Deposits
│  ├─ Upcoming Check-ins
│  ├─ Tentative Holds Expiring
│  ├─ Content / Rule Warnings
│  └─ Quick Actions
│
├─ Bookings
│  ├─ Inquiry List
│  ├─ Booking Pipeline
│  ├─ Booking Detail
│  ├─ Guest Record
│  ├─ Payment / Deposit Panel
│  ├─ Internal Notes
│  └─ WhatsApp Templates
│
├─ Calendar
├─ Rates & Packages
├─ Property Content
├─ Media
├─ Social & Reviews
├─ Rules & Policies
├─ Analytics
└─ Settings
```

The owner dashboard starts with Today, not generic analytics.

It should answer:

```text
What needs attention?
Who is coming?
Who has not paid?
Which inquiry needs reply?
Which rule/content/system issue may cause business damage?
```
