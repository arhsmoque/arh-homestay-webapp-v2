# Frontend Quality Checklist

## Accessibility

```text
keyboard complete
focus visible
form errors associated
no hover-only critical actions
touch targets comfortable
reduced motion respected
```

## State Coverage

```text
empty
loading
partial
success
error
validation_error
permission_denied
saving
retry
destructive_confirmation
recovery
```

## Responsiveness

```text
mobile owner Today view
booking detail mobile
calendar mobile
desktop operator sidebar
large-screen developer console
```

## Performance

```text
public site mostly static
media optimized
no autoplay-heavy homepage
operator console not loaded into public site
client JavaScript bounded
```
